像素格斗：觉醒擂台 - 跨平台打开说明

最简单的方法：
1. 不要只复制 index.html，要复制整个 pixel-fighters 文件夹。
2. Windows：双击 start-windows.bat，或用 Chrome / Edge 打开 index.html。
3. macOS：双击 start-mac.command，或用 Safari / Chrome 打开 index.html。
4. Linux：运行 start-linux.sh，或用浏览器打开 index.html。
5. 手机和平板：建议把游戏发布到网页地址后，用 Safari / Chrome / Edge 打开；页面内已经有触屏按钮。

支持的浏览器：
Chrome、Edge、Safari、Firefox 等现代浏览器。

离线和安装：
通过 http/https 地址打开时，游戏会注册离线缓存，并可作为 PWA 添加到桌面或手机主屏幕。
直接用 file:// 打开也能玩，但浏览器通常不会允许离线缓存安装。

想让 Google 搜到：
需要把这个文件夹发布到公网，例如 GitHub Pages、Netlify、Vercel 或自己的服务器。发布后搜索引擎才可能收录。
